App name:CareeronCSE 
Basic info:
This is an information app of computer science and engineering department.This app can help a student to know about all the fields of CSE for ensuring them to choose a right field for their career.

User Manual:
1.when you run this app ,firstly you can get some information about CSE and in the end of layout you will see a button "see fields" by clicking this you can go through to one activity to another another 
2.after clicking the button,we go to list of items activity where we describes items like software engineering,system analyst,Network engineering,Database administration,Web developer,Graphics designer and many more options we will use in future.
3.after clicking each button we will get information about that fields
So finally we can say this app is very friendly for a student and very helpful to get information about CSE .
